# OMNIA SSPs drivers generator

Code to extract population and GDP data following DIAMOND harmonizaton guidelines and obtain population, GDP and GDPpc drivers for the OMNIA model for SSPs scenarios.
- ```input``` folder contains data sources from IIASA, OECD, IMF and GCAM
- ```output``` folder contains the output drivers for OMNIA
- ```extract.py``` contains the code to extract the drivers
- ```DIAMOND - harmonisation protocol_18112024.docx``` contains the guidelines for the DIAMOND project as of November 2024.
